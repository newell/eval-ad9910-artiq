# Control AD9910 Evaluation Board with ARTIQ

The goal of this project is to control the [EVAL-AD9910](https://www.analog.com/en/resources/evaluation-hardware-and-software/evaluation-boards-kits/eval-ad9910.html) using [ARTIQ](https://github.com/m-labs/artiq) via the [EBAZ4205](https://github.com/xjtuecho/EBAZ4205).
